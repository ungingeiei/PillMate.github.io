PillMate

ARDUINO PROJECT

โปรเจกต์ IoT Microcontroller สำหรับการปล่อยยาอัตโนมัติที่สามารถตั้งเวลาได้ เป็นโปรเจคที่พัฒนาขึ้นเพื่อช่วยให้ผู้ป่วยสามารถรับประทานยาได้ตรงเวลาโดยไม่ต้องพึ่งการจดจำด้วยตนเอง ระบบจะใช้ ไมโครคอนโทรลเลอร์ เป็นตัวควบคุมหลักในการตั้งเวลา ตรวจสอบ และสั่งการให้ ตู้ปล่อยยา ทำงานตามเวลาที่กำหนด พร้อมทั้งมีระบบ แจ้งเตือนด้วยเสียงหรือไฟ เพื่อเตือนผู้ใช้เมื่อถึงเวลารับประทานยา

Features

ระบบจ่ายยาอัตโนมัติ

ตู้สามารถปล่อยยาได้โดยอัตโนมัติตามเวลาที่ตั้งไว้ล่วงหน้า ช่วยลดความเสี่ยงจากการลืมหรือกินยาผิดเวลา

แจ้งเตือนแบบเรียลไทม์

รองรับการเชื่อมต่อผ่าน IoT พร้อมระบบแจ้งเตือนด้วยเสียงเมื่อถึงเวลารับประทานยา

อินเทอร์เฟซใช้งานง่าย

ผู้ใช้สามารถตั้งเวลา และจัดการรอบการจ่ายยาผ่านแอปพลิเคชันได้อย่างสะดวก

Project details

โปรเจกต์นี้เป็นการใช้ ไมโครคอนโทรลเลอร์ ในการควบคุม เครื่องจ่ายยาอัตโนมัติ โดยใช้ Servo Motor เป็นตัวเปิดและปิดช่องเก็บยา เพื่อปล่อยยาในปริมาณและเวลาที่กำหนด

Main function

ปล่อยยาอัตโนมัติตามเวลาที่ตั้งไว้ พร้อมแจ้งเตือนผู้ใช้ด้วยเสียงเมื่อถึงเวลารับประทานยา
